# Apple-Stock-Prediction-XGBoost
A hybrid machine learning model predicting Apple (AAPL) stock direction using XGBoost, Technical Indicators (RSI, MACD, Bollinger Bands), and News Sentiment Analysis. Achieved ~58.5% accuracy.
Apple Stock Trend Prediction with XGBoost

Bu proje, Apple (AAPL) hisse senetlerinin günlük kapanış fiyatının bir sonraki gün **Yükseleceğini mi** yoksa **Düşeceğini mi** tahmin eden bir Makine Öğrenmesi modelidir.

##  Proje Hakkında
- **Algoritma:** XGBoost Classifier
- **Başarı Oranı:** %58.52 (Accuracy)
- **Veri Seti:** 2016-2024 yılları arası Yahoo Finance verileri + Haber Duygu Analizi (Sentiment)
- **Özellikler (Features):** RSI, MACD, Bollinger Bantları, SMA, Haber Skoru.
